Exportformate

Exportieren Sie nach PNG, Office, SVG, HTML und speziell für die Verwendung auf der mebis Lernplattform:

  • PNG: Optimal für schnelle Verarbeitung in Office-Anwendungen und im Internet.
  • Office: Kopie in der Zwischenablage zur Weiterverarbeitung in einer Textverarbeitung oder Tabellenkalkulation.
  • SVG: Eine Vektordatei, die nachträglich bearbeitet werden kann, beispielsweise mit dem kostenlosen Programm Inkscape.
  • HTML (für Profis): Geeignet für direktes Einbinden in HTML-Code, beispielsweise inline in einem mebis/Moodlekurs mit der Möglichkeit zum Skripting. Ein praktisches Beispiel dafür ist die Verwendung des aktuellen Jahres im Rechnungsdatum, ohne dass Sie sich um die Aktualisierung der Belege kümmern müssen. Hinweis: Von manchen Plattformen könnte Scripting in einer SVG nicht zugelassen sein oder in Zukunft deaktiviert werden.

Soweit nicht anders gekennzeichnet unterliegen alle per Button exportierbaren Inhalte der Lizenz CC0 1.0 Deed. Der Quellcode sowie Möglichkeiten zur Mitwirkung sind über das Repository auf Github zu erreichen.
