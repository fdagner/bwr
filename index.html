Häufig gestellte Fragen (FAQ)

Wo werden meine Daten gespeichert?

Alle Funktionen dieser Webanwendung laufen vollständig lokal in Ihrem Browser.

Das bedeutet:

  • ✓ Alle Daten (Modellunternehmen, Belege, Einstellungen) bleiben auf Ihrem Gerät
  • ✓ Es werden keine Daten an einen Server übertragen
  • ✓ Vollständige Kontrolle und Privatsphäre
  • ⚠ Daten gehen beim Löschen des Browser-Caches verloren
  • ⚠ Daten sind nicht automatisch auf anderen Geräten oder Browsern verfügbar

Empfehlung: Exportieren Sie regelmäßig Ihre Modellunternehmen als .yml-Datei, um ein Backup zu haben!

Wie setze ich ein Modellunternehmen als "Mein Unternehmen"?
  1. Wählen Sie aus dem Dropdown-Menü "Alle verfügbaren Unternehmen" ein Unternehmen aus
  2. Prüfen Sie die Unternehmensdetails in der Vorschau
  3. Klicken Sie auf "★ Als Modellunternehmen verwenden"
  4. Das Unternehmen wird im Browser gespeichert und bei der Belegerstellung automatisch verwendet

Tipp: Sie können jederzeit ein anderes Unternehmen auswählen oder das aktuelle mit "✕ Entfernen" löschen.

Wie kann ich Modellunternehmen exportieren und importieren?

Exportieren:

  1. Klappen Sie den Bereich "Unternehmen importieren/exportieren" auf
  2. Klicken SIe auf "Aktuelle Unternehmen exportieren"

Importieren:

  1. Ziehen Sie eine .yml-Datei per Drag & Drop in den Upload-Bereich
  2. Oder klicken Sie auf den Bereich und wählen Sie eine Datei aus
  3. Die Unternehmen werden importiert und stehen sofort zur Verfügung

Praktisch für: Datensicherung, Nutzung auf mehreren Geräten, Austausch mit Kollegen

Wie erstelle ich ein eigenes Modellunternehmen?
  1. Klappen Sie den Bereich "Unternehmen manuell hinzufügen" auf
  2. Füllen Sie alle Felder (*) aus:
    • Basisdaten (Name, Rechtsform, Branche, Motto, Inhaber)
    • Adresse & Kontakt (Straße, PLZ, Ort, Telefon, E-Mail, Internet)
    • Finanz- & Steuerdaten (Bank, IBAN, BIC, USt-IdNr., Steuernummer, Akzentfarbe)
  3. Optional: Laden Sie ein Logo als SVG-Datei hoch (max. 500KB)
  4. Klicken Sie auf "✓ Unternehmen hinzufügen"
Auf welche Weise können Belege erstellt werden?

Manuelle Belege:

  • Werden unter dem Menüpunkt "Belege" erstellt
  • Sie geben alle Informationen manuell ein (Datum, Beträge, Positionen etc.)
  • Ideal für: Einzelne Übungsbelege, individuelle Szenarien, freie Gestaltung

Verlinkte Belege:

  • Werden unter Menüpunkten wie zum Beispiel "Einkauf" oder "Verkauf" erstellt
  • Basieren auf zusammenhängenden Geschäftsvorfällen
  • Belege werden automatisch miteinander verknüpft (z. B. Geschäftsfall → Rechnung)
Welches Exportformat soll ich verwenden?

PNG:

  • Schnelle Integration
  • H5P-Aktivitäten in der ByCS Lernplattform
  • Wenn keine nachträgliche Bearbeitung nötig ist

Office:

  • Direktes Einfügen in Textverarbeitung, Tabellenkalkulation...
  • Wenn Sie die Belege in Office-Dokumenten weiterbearbeiten möchten

SVG:

  • Nachträgliche Bearbeitung (z. B. unter "Belege importieren" oder mit Inkscape)
  • Skalierbare Grafiken ohne Qualitätsverlust
  • ByCS Lernplattform (Text-/Medienfelder, Textseiten, Tests)

HTML/Inline SVG:

  • Fortgeschrittene Nutzer mit HTML-Kenntnissen
  • Dynamische Inhalte (z.B. automatisches Datum)
  • Direktes Einbinden in Moodle/ByCS-Kurse mit Scripting
Was kann ich tun, wenn meine Daten verloren gegangen sind?

Wenn Sie ein Backup haben:

  1. Öffnen Sie den Bereich "Unternehmen importieren/exportieren"
  2. Importieren Sie Ihre gesicherte .yml-Datei
  3. Ihre Modellunternehmen sind wiederhergestellt
